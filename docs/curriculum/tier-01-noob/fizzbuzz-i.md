## Task

- Create a file named fizzbuzz.py.
- For numbers 1..N:
- - Print Fizz if divisible by 3
- - Print Buzz if divisible by 5
- - Print FizzBuzz if divisible by both
- - Otherwise print the number
- (Optional) Let the user choose N; default to 100.
